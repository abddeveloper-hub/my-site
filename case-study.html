Case Study 01

Aperture

WebGL Three.js React

The Challenge

Aperture was conceived as a boundary-pushing photographic portfolio. The goal was to move away from standard masonry grids and build a fluid, 3D-accelerated gallery that mimics the optics of a physical camera lens.

Performance was a critical constraint. We needed to load high-resolution imagery and apply real-time depth-of-field shaders without dropping below 60FPS on mobile devices.

[ High Resolution UI Mockup ]

The Solution

Using Three.js and custom GLSL shaders, we built a virtual lens system. As users scroll, the images pass through a focal plane, coming into sharp, chromatic focus before blurring out into bokeh shapes.

We achieved optimal performance by utilizing InstancedMesh for the gallery items and compressing all textures via ASTC/ETC1 depending on the user's hardware profile.
